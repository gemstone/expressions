About the Gemstone Library

The Grid Protection Alliance Gemstone Library is a collection of handy open source .NET code assemblies based on continuous development since 2004..

These packages represent a focused re-envisioning of the best libraries, i.e., the "gems", from the Grid Solutions Framework (GSF), which has long been in use by various open source projects and electric power utilities worldwide.

The Gemstone Library contains a large variety of code useful for any .NET project with of hundreds of class libraries that extend or expand the functionality included in .NET, including many tools and protocol parsers that are useful for the electric power industry. For example:

  • gemstone.threading The Gemstone Threading Library organizes all Gemstone functionality related to threading.

  • gemstone.expressions The Gemstone Expressions Library organizes all Gemstone functionality related to expression parsing.
