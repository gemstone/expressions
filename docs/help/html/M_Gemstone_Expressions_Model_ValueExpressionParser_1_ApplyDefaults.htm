Generates a delegate that will update an instance of type T applying any specified DefaultValueAttribute or DefaultValueExpressionAttribute instances that are declared on the type T properties. Target T instance is accepted as the parameter to the returned delegate ActionT.

Remarks
This function is useful for generating a delegate to a compiled function that will update objects of type T where properties of the type of have been decorated with DefaultValueAttribute or DefaultValueExpressionAttribute attributes. The updated object will automatically have applied any defined default values as specified by the encountered attributes.
 Note
This function will assign evaluated expression values to properties in an existing model.
